2. Bot Purpose

Scythe Bot provides features including, but not limited to:

User verification system

Temporary voice channel creation and management

Moderation tools

3. Data Collected

The Bot may temporarily store the following data:

Discord User IDs (for verification, ticket authors, VC ownership, blocklists)

Passwords

Payment information

Private messages (DMs)
