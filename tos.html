Last Updated: January 2025

1. Acceptance of Terms

2. Bot Purpose

Scythe Bot provides features including, but not limited to:

  • Support ticket system
  • User verification system
  • Temporary voice channel creation and management
  • Moderation tools

3. Data Collected

The Bot may temporarily store the following data:

  • Discord User IDs (for verification, ticket authors, VC ownership, blocklists)
  • Guild IDs and role IDs (for configuration)
  • Message content inside tickets (for transcripts)

The Bot does NOT collect:

  • Email addresses
  • Passwords
  • Payment information
  • Private messages (DMs)

4. Data Storage Duration

Ticket transcripts are stored in the server’s configured transcript channel until manually deleted. VC ownership and blocklists persist until changed by the user. No data is shared with third parties.
