Scythe Bot – Privacy Policy

Last Updated: December 2025

1. Introduction

This Privacy Policy explains what information Scythe Bot (“the Bot”) collects, how it is used, and how it is stored.
By using the Bot, you agree to this policy. If you do not agree, remove the Bot from your server.

2. Information the Bot Collects

The Bot only collects information required for its features to operate correctly.

This includes:

a) Discord User Information

User IDs

Usernames (as provided by Discord)

Role IDs (for permissions, verification, or alerts)

b) Guild (Server) Information

Guild IDs

Channel IDs (tickets, logs, alerts, verification, VC management)

Role IDs (staff roles, verified roles, alert ping roles)

c) Feature-Specific Data

Ticket System

Temporary ticket messages and user responses

Optional transcripts posted to configured channels

Temporary Voice Channels

VC ownership data

Co-owner lists

VC bans and control settings

Verification System

Account age check results

Verification logs (posted only to configured channels)

YouTube & Twitch Alerts

YouTube channel IDs

Twitch channel/user IDs

Alert configuration (selected types, post channels, role mentions, templates)

Processed video or stream IDs to prevent duplicate alerts

3. Information the Bot Does NOT Collect

Scythe Bot does not collect or store:

Email addresses

Passwords

IP addresses

Payment information

Private messages (DMs)

Message content outside configured bot channels

YouTube or Twitch account credentials

The Bot only accesses data that Discord, YouTube, and Twitch APIs provide to all bots and applications.

4. How the Information Is Used

Collected information is used only to:

Manage temporary voice channels

Provide ticket support and generate transcripts

Verify members and assign roles

Deliver YouTube and Twitch alerts to configured channels

Prevent duplicate alerts for the same video or stream

Log moderation or system actions

The Bot does not sell, share, or transfer data to third parties.

5. Data Storage Duration
Temporary Voice Channel Data

Stored only while the VC exists

Removed automatically when the VC is deleted

Ticket Data

Transcripts exist only in your server’s configured transcript channel

The Bot does not store transcripts elsewhere

Verification Logs

Stored only in your configured verification log channel

Alert Configuration Data

Alert settings (channels, roles, templates) are stored until removed by server administrators

6. Data Removal

Users may request removal of personal data from:

VC blocklists

Ticket transcripts (via server admins)

Verification logs (via server admins)

Removing the Bot from a server immediately stops all data collection and processing.

7. Security

No data is sold or transferred externally

Configuration data is stored locally on the Bot owner’s secure VPS

Only the Bot owner has server-level file access

Server admins cannot access raw storage or databases

8. Children’s Privacy

Scythe Bot does not knowingly collect data from users under the age of 13.
(Discord itself requires users to be 13 or older.)

9. Third-Party Services

The Bot communicates with:

Discord’s API

YouTube’s public Data API

Twitch’s public API

These services are used only to fetch public content (videos or live status).
No personal user data is shared with these services.
